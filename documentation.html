Installation

Install MotionKit using npm or yarn:

npm install motionkit

Or with yarn:

yarn add motionkit

Quick Start

Here's how to get started with MotionKit:

import { Button, Card, Input } from 'motionkit';

function App() {
  return (
    <div>
      <Button variant="default">Click me</Button>

      <Card>
        <Card.Header>
          <Card.Title>Card Title</Card.Title>
        </Card.Header>
        <Card.Content>
          <p>Card content</p>
          <Input placeholder="Enter text..." />
        </Card.Content>
      </Card>
    </div>
  );
}

Components

Button

A customizable button component with multiple variants and animations. Now includes improved ripple effect and `animateOn` prop for entry animations.

Props:

  • variant: "default" | "destructive" | "outline" | "ghost"
  • size: "sm" | "md" | "lg"
  • ripple: boolean - Enable ripple effect (default: true)
  • loading: boolean - Show loading state
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
<Button variant="default" size="md" animateOn={{ type: 'fadeIn' }}>Default Button</Button>

Card

A container component with header, title, and content sections. Supports animated entrance, staggered children, and `animateOn` prop.

Props:

  • variant: "default" | "elevated" | "glass"
  • hover: "none" | "lift" | "glow"
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
  • staggerChildren: boolean - Enables staggered animation for direct children (Framer Motion)
<Card variant="default" hover="lift" animateOn={{ type: 'slideUp' }}>
  <Card.Header>
    <Card.Title>Card Title</Card.Title>
  </Card.Header>
  <Card.Content>
    <p>Card content</p>
  </Card.Content>
</Card>

Input

A styled input field component with variants, sizes, and `animateOn` prop.

Props:

  • variant: "default" | "error" | "success" - Visual state of the input
  • size: "sm" | "md" | "lg" - Size of the input field
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
<Input placeholder="Enter text..." variant="default" size="md" animateOn={{ type: 'fadeIn' }} />
<Input placeholder="Error state" variant="error" />

Modal

An animated modal dialog component with improved accessibility and control.

Props:

  • isOpen: boolean - Control modal visibility
  • onClose: () => void - Callback for closing modal (also triggered by Escape key and overlay click)

Features: Escape key close, basic focus management, scroll locking on body.

<Modal isOpen={isOpen} onClose={() => setIsOpen(false)}>
  <Card.Header>
    <Card.Title>Modal Title</Card.Title>
  </Card.Header>
  <Card.Content>
    <p>Modal content</p>
  </Card.Content>
</Modal>

Toggle

A switch component with smooth animation, customizable colors, and label support.

Props:

  • enabled: boolean - Current state of toggle
  • onChange: (enabled: boolean) => void - Callback for state changes
  • size: "sm" | "md" | "lg"
  • label: React.ReactNode - Optional label for the toggle
  • activeColor: string - Tailwind class for active background color (default: bg-primary-500)
  • inactiveColor: string - Tailwind class for inactive background color (default: bg-gray-200)
<Toggle enabled={enabled} onChange={setEnabled} size="md" label="Enable feature" />

Avatar

A component to display user profile pictures or fallback initials. Now includes `animateOn` prop.

Props:

  • src: string - Image source URL
  • fallback: string - Text to show when no image
  • size: "sm" | "md" | "lg" | "xl"
  • shape: "circle" | "square"
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
<Avatar src="/path/to/image.jpg" fallback="JD" size="md" animateOn={{ type: 'scale' }} />

Badge

A small countable component for displaying notifications or labels. Now includes `animateOn` prop.

Props:

  • variant: "default" | "primary" | "secondary" | "destructive" | "outline" | "ghost"
  • size: "sm" | "md" | "lg"
  • rounded: "sm" | "md" | "lg" | "full"
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
<Badge variant="primary" size="md" animateOn={{ type: 'bounce' }}>New</Badge>

Alert

A component to display important messages to the user. Now includes default icons and improved accessibility.

Props:

  • variant: "default" | "info" | "success" | "warning" | "destructive"
  • show: boolean - Control alert visibility
  • closable: boolean - Show close button
  • onClose: () => void - Callback for closing
  • title: string - Alert title
  • description: string - Alert description
  • icon: React.ReactNode - Custom icon (overrides default)
<Alert
  variant="info"
  title="Info Alert"
  description="This is an info message"
  closable
  onClose={() => setShowAlert(false)}
/>

Checkbox

A customizable checkbox input with animated states and label support.

Props:

  • checked: boolean - Current checked state
  • onCheckedChange: (checked: boolean) => void - Callback for state changes
  • disabled: boolean - Disable checkbox
  • size: "sm" | "md" | "lg"
  • label: React.ReactNode - Optional label for the checkbox
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
<Checkbox checked={isChecked} onCheckedChange={setIsChecked} label="Remember me" animateOn={{ type: 'slideUp' }} />

Radio Group

A group of radio buttons where only one option can be selected at a time.

Props (RadioGroup):

  • value: string - Currently selected value
  • onValueChange: (value: string) => void - Callback for value changes
  • disabled: boolean - Disable the entire group
  • orientation: "horizontal" | "vertical" - Layout of radio items
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)

Props (RadioGroupItem):

  • value: string - Value of this radio item
  • id: string - Unique ID
  • disabled: boolean - Disable this item
<RadioGroup value={selectedValue} onValueChange={setSelectedValue} orientation="horizontal" animateOn={{ type: 'scale' }}>
  <RadioGroupItem value="option-a">Option A</RadioGroupItem>
  <RadioGroupItem value="option-b">Option B</RadioGroupItem>
</RadioGroup>

Select

A customizable dropdown/select component for choosing an option from a list.

Props (Select):

  • value: string - Currently selected value
  • onValueChange: (value: string) => void - Callback for value changes
  • disabled: boolean - Disable the select
  • placeholder: string - Placeholder text

Props (SelectItem):

  • value: string - Value of this item
  • disabled: boolean - Disable this item
<Select value={selectedFruit} onValueChange={setSelectedFruit} placeholder="Choose a fruit">
  <SelectTrigger animateOn={{ type: 'fadeIn' }} />
  <SelectContent>
    <SelectItem value="apple">Apple</SelectItem>
    <SelectItem value="banana">Banana</SelectItem>
    <SelectSeparator />
    <SelectItem value="orange">Orange</SelectItem>
  </SelectContent>
</Select>

Tabs

A set of layered content areas, with only one visible at a time.

Props (Tabs):

  • defaultValue: string - Initial active tab
  • value: string - Controlled active tab
  • onValueChange: (value: string) => void - Callback for tab changes
  • orientation: "horizontal" | "vertical" - Layout of tabs
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)

Props (TabsTrigger):

  • value: string - Value of this tab
  • disabled: boolean - Disable this tab

Props (TabsContent):

  • value: string - Value corresponding to the tab
<Tabs defaultValue="tab1" animateOn={{ type: 'slideDown' }}>
  <TabsList>
    <TabsTrigger value="tab1">Account</TabsTrigger>
    <TabsTrigger value="tab2">Password</TabsTrigger>
  </TabsList>
  <TabsContent value="tab1">Make changes to your account here.</TabsContent>
  <TabsContent value="tab2">Change your password here.</TabsContent>
</Tabs>

Tooltip

A small, contextual popup that displays information about an element on hover or focus.

Props (Tooltip):

  • delayDuration: number - Delay before showing/hiding (ms)
  • open: boolean - Controlled open state
  • onOpenChange: (open: boolean) => void - Callback for open state changes

Props (TooltipContent):

  • side: "top" | "bottom" | "left" | "right" - Side to display tooltip
  • align: "start" | "center" | "end" - Alignment of tooltip
  • offset: number - Offset from trigger (px)
<Tooltip>
  <TooltipTrigger>
    <Button variant="outline">Hover me</Button>
  </TooltipTrigger>
  <TooltipContent side="right">
    Hello from Tooltip!
  </TooltipContent>
</Tooltip>

Progress Bar

A visual indicator of progress toward the completion of a task.

Props:

  • value: number - Current progress (0-100)
  • max: number - Maximum value (default: 100)
  • animateOn: AnimationConfig - Configuration for entry animations (Anime.js)
  • indicatorClassName: string - Tailwind classes for the progress indicator
<ProgressBar value={75} animateOn={{ type: 'fadeIn' }} />

Spinner

A visual indicator for loading states with customizable size and color.

Props:

  • size: "sm" | "md" | "lg" | "xl"
  • color: string - CSS color value (e.g., "#0ea5e9" or "currentColor")
<Spinner size="lg" color="currentColor" />
